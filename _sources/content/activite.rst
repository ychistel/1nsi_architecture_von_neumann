Alice et Bob souhaitent acheter chacun un ordinateur portable.

1. Si vous deviez acheter un ordinateur portable, à quelles caractéristiques techniques feriez-vous attention ?
2. Comparer les deux modèles retenus par Alice et Bob:

   - Alice a repéré un modèle dans une grande surface visible en ligne : `pc-lenovo`_
   - Bob en a repéré un autre : `pc-hp`_

3. Le site `Versus`_ a comparé les deux processeurs. Quelle est sa conclusion ?
4. Les ordinateurs actuels ont une architecture selon le modèle de "Von Neumann".
